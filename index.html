About Web Application

Web Application - is a small and simple GTK WebKit based application, which allows you to create so-called Web applications, similar of created Epiphany browser or Internet explorer in Windows 7 and above.

To create an application you can simply enter URL, title web-applications and an optional description. Additionally, you can choose the image that will be an application icon.

Applications are stored in a separate directories. Each application is isolated and keeps all the necessary information, including cookies, application settings and saved state.

After creating the application, the application shortcut is automatically copied to the main menu of the system and is available in the group "Web applications".

At this moment, Web application is in beta testing. Any comments and suggestions are welcome.

Install

To install the Web Application download the current version of the application. Unzip it. In a root of the unpacked folder there is a build.sh file. Obtain root access and run:
./build.sh install

You also can create an archive with an application for further create a package for your Linux distribution.

For this in a root of the unpacked folder there is a build.sh file. Launch it without parameters:
./build.sh
as a result, in the dist folder the archive with application will be saved.
